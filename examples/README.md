# EasyPygame Examples

Example games demonstrating EasyPygame features.

## Running Examples

Make sure EasyPygame is installed:
```bash
pip install EasyPygame
# or from source:
pip install -e .
```

Then run any example:
```bash
python examples/clicker_game.py
python examples/shooter_demo.py
```
